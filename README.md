# Ludic Homeostasis Model (LHM) v0.9-beta

LHM — инженерный инструмент для навигации в хаосе геймдева.

Модель описывает игру как инструмент регуляции состояния игрока. Она помогает формулировать обещание, понимать аудиторию и проектировать core loop так, чтобы снижать риск диссонанса ожиданий.

Это **бета-версия**. Формулировки, структура и отдельные части будут уточняться по мере тестов на проектах и обратной связи.

**English**: 
LHM is an engineering-oriented framework for game designers: it treats games as tools for regulating a player’s state and helps you reason about audience fit, promise, and core loop coherence.

If you’re new here, start with [`LHM.md`](./LHM.md). English translation is work-in-progress.
